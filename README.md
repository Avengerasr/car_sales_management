# car_sales_management
DBMS Mini Project - Car Sales Management System

A simple web-based **Car Sales Management System** built using **Python (Flask)** and **MySQL**.  
It allows you to **add, view, update, and delete** car sales records easily through a user-friendly dashboard.

# Features

- Add new car sales details  
- View all sales in a clean table format  
- Edit or delete existing records  
- Real-time data stored in MySQL database  
- Simple and modern HTML + CSS interface
