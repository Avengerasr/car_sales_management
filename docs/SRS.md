# Software Requirement Specification (SRS)

## 1. Introduction
The Car Sales Management System automates dealership operations by managing car inventory, customer details, and sales records.

## 2. Functional Requirements
- Add, view, update, and delete cars.
- Register customers and record sales.
- Generate sales summary reports.

## 3. Non-Functional Requirements
- Easy to use web interface.
- MySQL backend.
- Runs on local Flask server.

## 4. Constraints
- Works locally, requires Python and MySQL.
